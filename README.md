# CAPS Grade 12 Educational App 🎓

A comprehensive educational web application designed specifically for South African Grade 12 learners following the CAPS (Curriculum and Assessment Policy Statement) curriculum. This app provides structured learning materials, progress tracking, and interactive features for seven core subjects.

## 📚 Subjects Covered

The app includes comprehensive content for the following CAPS Grade 12 subjects:

1. **Mathematics** - Pure Mathematics with advanced concepts
2. **Mathematics Literacy** - Practical mathematical applications
3. **Life Science** - Biology and life processes
4. **Accounting** - Financial management and reporting
5. **Business Studies** - Business principles and management
6. **History** - South African and world history
7. **Economics** - Economic principles and systems

## ✨ Features

### 🎯 Core Features
- **Interactive Dashboard** - Overview of learning progress and quick subject access
- **Subject Navigation** - Easy switching between different subjects
- **Topic Organization** - Each subject divided into comprehensive topic areas
- **Progress Tracking** - Visual progress bars and completion statistics
- **Local Storage** - Progress saved locally in browser
- **Responsive Design** - Works on desktop, tablet, and mobile devices

### 🎨 User Experience
- **Modern UI** - Beautiful gradient design with smooth animations
- **Interactive Elements** - Hover effects and click animations
- **Mobile-Friendly** - Responsive navigation with mobile menu
- **Visual Feedback** - Notifications and completion indicators
- **Smooth Transitions** - Animated section switching and scroll effects

### 📊 Learning Management
- **Topic Completion** - Click topics to mark as completed
- **Progress Visualization** - Real-time progress bars for each subject
- **Overall Statistics** - Dashboard showing total completion percentage
- **Persistent Progress** - Progress saved between sessions

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. **Clone or Download** the repository
2. **Open** `index.html` in your web browser
3. **Start Learning** - No server setup required!

### File Structure

## 🎨 Customization

### Color Scheme
The app uses a beautiful purple gradient theme that can be customized by modifying CSS variables:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f8f9ff;
}
```

### Adding Content
To add new topics or subjects:
1. Update the HTML structure in `index.html`
2. Add corresponding styles in `style.css`
3. Update JavaScript arrays in `script.js`
